interface d'administration robomongo

Objectif : tester l'interface d'administration robomongo

robomongo est une interface graphique permettant de piloter des base de données de type MongoDB. Dans cet exercice nous allons découvrir le fonctionnement de cette interface après l'avoir installée à partir du site officiel. Pour cela nous procéderons ainsi :

  1. à partir de l'archive PHP robomongo.org/files/linux/robomongo-0.8.5-x86_64.rpm destinée aux distributions de type RedHat nous réaliserons l'installation du système d'administration Robomongo.
  2. nos Testerons ensuite cette interface sur l'instance de mongod en service.
  1. Pour réaliser l'installation exécuter la commande suivante : 
    rpm -ivh http://robomongo.org/files/linux/robomongo-0.8.5-x86_64.rpm
  2. Pour tester l'interface exécuter la commande suivante : 
    robomongo
